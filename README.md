# Random quote machine

> Random tweetable quotes from famous people. Free Code Camp exercise task. 

## Start

```sh
$ npm install
$ npm start
```

Also, you can configure server port using `PORT` environment variable.

```sh
$ PORT=5000 npm start
```

## Testing

```sh
$ npm test
```
